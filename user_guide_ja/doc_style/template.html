Foo クラス

Foo クラスの簡単な説明です。 もしも CodeIgniter 固有のクラスを継承している場合には、その CodeIgniter クラスのドキュメントへのリンクをここに貼ります。

重要:  ここには重要事項を 強調 して書きます。

特徴:

  • Foo
  • Bar

使用法の見出し

Foo の使用法を詳しく書きます。変数やファイル名をハイライトとともに普通のテキストで <var> タグ で書きます。そしてコードを <dfn> タグ で書きます。

小見出し

サンプルコードは <code></code> タグ内に記述します:

$this->load->library('foo');

$this->foo->bar('bat');

設定項目の表

表をつかって長い設定項目のリストを書きます。

設定項目 初期値 Value 選択肢 説明
foo Foo なし foo の詳細。
bar Bar bat, bag, または bak bar の詳細。

Foo メソッドのリファレンス

$this->foo->bar()

詳細

$this->foo->bar('baz')
